# 项目介绍

- 基于Asp .net core 3.1的仿携程网的Restful web api 项目
- 技术栈
  - Asp .net core 3.1
  - Entity Framework core
  - MySQL
  - AutoMapper
  - Jwt
  - Asp .net core Identity 
  - Nlog
  - NewtonsolftJson
- 功能
  - 旅游路线增删改查
  - Jwt用户权限授权认证
  - 购物车系统
  - 订单系统
  - 数据分页
  - 数据排序
  - 数据塑形
